# Purple

**Purple** is a landing project based on **Vue 3 + Vite**, built using **TailwindCSS**.  
The project was created for the practice of modern UI, adaptive layout and the component approach of Vue. I am actively practicing **TailwindCSS** for personal experience

---

## Tech Stack

- **Vue 3** (Composition API, `<script setup>`)
- **Vite**
- **TailwindCSS**
- Node.js / npm

---

## Installation and launch

Clone project:

```bash
git clone https://github.com/Sh0kerbled/purple.git
cd purple
npm install
```
